# Employee Management System

A full-stack Employee Management System built using **Angular 17**, **Spring Boot**, and **MySQL (XAMPP)**.

## 🚀 Features
- Full CRUD operations for employees
- Server-side pagination
- Search by name and filter by department
- Backend validation and global exception handling
- Angular reactive forms with real-time validation
- RESTful API integration

## 🛠 Tech Stack
- Frontend: Angular 17 (Standalone Components)
- Backend: Spring Boot, Spring Data JPA
- Database: MySQL (XAMPP)
- Tools: Postman, GitHub
